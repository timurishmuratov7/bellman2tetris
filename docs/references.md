# References

Keep this list short. You do not need twenty tabs open to learn RL well.

## Core RL

- Sutton and Barto, *Reinforcement Learning: An Introduction*:
  [book page](https://incompleteideas.net/publications.html)
  and [second edition draft PDF](https://www.incompleteideas.net/book/bookdraft2018mar21.pdf)
- OpenAI Spinning Up:
  [Key Concepts in RL](https://spinningup.openai.com/en/latest/spinningup/rl_intro.html)
- David Silver's RL course:
  [teaching page](https://www.davidsilver.uk/teaching/)
  and [Lecture 1 PDF](https://www.davidsilver.uk/wp-content/uploads/2020/03/intro_RL.pdf)
